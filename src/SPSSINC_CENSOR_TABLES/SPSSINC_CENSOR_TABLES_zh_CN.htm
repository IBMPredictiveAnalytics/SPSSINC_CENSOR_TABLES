检剔表

通过使用此对话框，可以在透视表中按某个指定统计的值删除或隐藏特定单元格。例如，如果单元格计数小于 5，那么可以删除平均值统计。

在检剔统计标签字段中，针对包含检剔统计的行或列输入最内级别标签的文本。文本大小写必须匹配，并且必须包括该标签中的所有空格。

从下拉列表中选择要执行的比较类型。执行的检验是
abs(单元格值) testtype critvalue。
例如，abs(单元格计数) < 5。

在要检剔的相邻行或列字段中列出要检剔的单元格。要检剔的单元格由它们相对于检剔统计的位置指示。负数表示位于检剔单元格的左侧或上方，而正数表示位于该单元格的右侧或下方。在列表中包括 0 可以对检剔统计本身进行检剔。

例如，如果表中含有列计数、平均值和标准差，那么可以通过在列表中输入
1 2
，根据计数对平均值和标准差进行检剔。

可以在表 OMS 子类型字段中指定所要处理的表的 OMS 子类型。您可以通过在大纲中右键单击实例或根据 Utilties/OMS 标识确定子类型。大小写和空格无关紧要。

缺省情况下，在查看器中将对最新的透视表进行检剔。您可以指定 ALL 以检剔所有表。如果指定了 OMS 表子类型，那么检剔限制于该类型的表。

选中隐藏准则行或列框可以隐藏包含检剔统计的整行或整列。

缺省情况下，将添加文字说明文本，以显示检剔单元格数以及检剔统计的名称。取消选中添加含有检剔信息的文字说明可以阻止上述操作。

通过选择相应的单选按钮，可以控制是始终显示此文字说明，还是仅在实际检剔单元格后才显示。

选中定制文字说明文本，然后在下面输入文本以指定定制文字说明文本。

缺省情况下，检剔的单元格将显示为空。通过在检剔的单元格的符号中输入由一个或多个字符构成的字符串，可以指定另一替换文本。

必备软件

需要 Python Programmability 插件。您可以使用 SPSS Statistics 安装 CD 来安装此插件，也可以从 SPSS Developer Central (www.spss.com/devcentral) 下载此插件。

通过语法提供的其他功能

执行 SPSSINC CENSOR TABLES /HELP 可以生成完整的语法图表。可以在语法中包括所要执行的命令，从而在 SPSSINC CENSOR TABLES 命令中生成表。在表生成命令失败的情况下，此功能可以提供更完善的控制。
