Tabelle zensieren

Verwenden Sie dieses Dialogfeld, um bestimmte Zellen in einer Pivot-Tabelle basierend auf dem Wert einer angegebenen Statistik in der Tabelle mit Leerzeichen zu überschreiben oder auf andere Weise unkenntlich zu machen. Sie könnten z. B. die MEAN-Statistik mit Leerzeichen überschreiben, wenn die Zellenanzahl unter 5 liegt.

Geben Sie in das Feld Beschriftung der Zensurstatistik den Text der innersten Ebene der Beschriftung für die Zeile oder Spalte ein, die die Zensurstatistik enthält. Sie müssen die Groß-/Kleinschreibung des Texts beachten und ggf. Leerzeichen in die Beschriftung einschließen.

Wählen Sie den durchzuführenden Typ von Vergleich aus der Dropdown-Liste aus. Durchgeführter Test:
Abs(Zellenwert) Testtyp Testwert.
Beispiel: Abs(Zellenanzahl) < 5.

Listen Sie die zu zensierenden Zellen im Benachbarte zu zensierende Zeilen oder Spalten auf. Die zu zensierenden Zellen werden durch ihre Position relativ zur Zensurstatistik angegeben. Negative Zahlen befinden sich links oder über der Zensurzelle, positive Zahlen rechts oder unter dieser Zelle. Schließen Sie 0 in die Liste ein, um die Zensurstatistik selbst zu zensieren.

Wenn eine Tabelle z. B. Spaltenanzahl, Mittelwert und Standardabweichung hat, könnten Sie den Mittelwert und die Standardabweichung basierend auf der Anzahl zensieren, indem Sie
1 2
in die Liste eingeben.

Sie können den zu verarbeitenden OMS-Tabellenuntertyp im Feld OMS-Tabellenuntertyp angeben. Sie können den Untertyp ermitteln, indem Sie mit der rechten Maustaste in die Gliederung auf einer Instanz klicken oder indem Sie Extras/OMS-IDs auswählen. Groß-/Kleinschreibung und Leerzeichen sind nicht von Bedeutung.

Standardmäßig wird die Zensur auf die letzte Pivot-Tabelle im Viewer angewendet. Sie können Alle angeben, um alle Tabellen zu zensieren. Wenn ein OMS-Tabellenuntertyp angegeben wird, wird die Zensur auf diesen Tabellentyp begrenzt.

Wählen Sie das Feld Kriteriumszeile oder -spalte ausblenden aus, um die gesamte Zeile oder Spalte mit der Zensurstatistik auszublenden.

Standardmäßig wird Titelzeilentext hinzugefügt, der die Anzahl der zensierten Zellen und den Namen der Zensurstatistik angibt. Wählen Sie Titelzeile mit Zensurinformationen hinzufügen ab, um dies zu verhindern.

Sie können festlegen, ob die Titelzeile immer angezeigt wird oder nur, wenn tatsächlich Zellen zensiert wurden. Wählen Sie dazu das entsprechende Optionsfeld aus.

Wählen Sie Angepasster Titelzeilentext aus und geben Sie unten den Text ein, der als benutzerdefinierter Titelzeilentext verwendet werden soll.

Standardmäßig werden zensierte Zellen leer dargestellt. Geben Sie ein Zeichen oder eine Zeichenfolge in Symbol für zensierte Zellen ein, um einen anderen Ersetzungstext festzulegen.

Voraussetzungen

Das Python-Programmierungs-Plug-in ist erforderlich. Es kann von der SPSS Statistics-Installations-CD installiert oder von SPSS Developer Central (www.spss.com/devcentral) heruntergeladen werden.

Weitere über Syntax verfügbare Funktionen

Die Ausführung von SPSSINC CENSOR TABLES /HELP erzeugt ein Diagramm der vollständigen Syntax. In der Syntax können Sie den Befehl einschließen, der im Befehl SPSSINC CENSOR TABLES ausgeführt werden soll, um die Tabelle zu erstellen. Dies bietet bessere Kontrolle, wenn der Befehl zur Tabellenerstellung fehlschlägt.
