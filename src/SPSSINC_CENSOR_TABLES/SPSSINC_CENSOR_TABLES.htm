Censor Table

Use this dialog to blank out or otherwise obscure certain cells in a pivot table based on the value of a specified statistic in the table. For example, you could blank out the mean statistic if the cell count is less than 5.

In the field Label of Censoring Statistic, enter the text of the innermost level of the label for the row or column containing the censoring statistic. You must match the case of the text and include any white space in the label.

Select the type of comparison to be done from the dropdown list. The test carried out is
abs(cell value) testtype critvalue.
For example, abs(cell count) < 5.

List the cells to be censored in the Neighboring rows or columns to censor field. The cells to be censored are indicated by their position relative to the censoring statistic. Negative numbers are to the left or above the censoring cell, and positive numbers are to the right or below that cell. Include 0 in the list to censor the censoring statistic itself.

For example, if a table has columns count, mean, and standard deviation, you could censor the mean and standard deviation based on the count by entering
1 2
in the list.

You can specify the OMS subtype of the table to process in the Table OMS Subtype field. You can find the subtype by right clicking in the outline on an instance or from Utilties/OMS identifiers. Case and white space do not matter.

By default, the censoring will be applied to the last pivot table in the Viewer. You can specify All to censor all tables. If an OMS table subtype is specified, the censoring is limited to tables of that type.

Check the box Hide criterion row or column to hide the entire row or column containing the censoring statistic.

By default, caption text is added showing the number of cells censored and the name of the censoring statistic. Uncheck Add caption with censoring information to prevent this.

You can control whether the caption always appears or only if any cells are actually censored by choosing the appropriate radio button.

Check Custom caption text and enter text below to have custom caption text.

By default, censored cells are made blank. Enter a string of one or more characters in Symbol for Censored Cells to specify different replacement text.

Prerequisites

The Python Programmability plug-in is required. It can be installed from the SPSS Statistics installation cd or downloaded from SPSS Developer Central (www.spss.com/devcentral).

Additional Features Available Through Syntax

Executing SPSSINC CENSOR TABLES /HELP produces a complete syntax chart. In syntax you can include the command to be executed to produce the table within the SPSSINC CENSOR TABLES command. This give better control if the table-generating command fails.
