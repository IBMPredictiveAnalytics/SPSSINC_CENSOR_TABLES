Цензурировать таблицу

Используйте это диалоговое окно, чтобы отключить или иным способом игнорировать ячейки сводной таблицы выбранного типа на основании указанного статистического показателя в таблице. Например, можно отключить статистику средних значений, если количество в ячейке меньше пяти.

В поле Метка статистики цензурирования введите текст метки самого внутреннего уровня вложения для строки или столбца, содержащего статистику цензурирования. Вы должны согласовать наблюдение для этого текста и включить в метку какие-то пробельные символы.

В выпадающем списке выберите тип сравнения для использования. Проводимая проверка - это
abs(значение для ячейки) тип_проверки критическое_значение.
Например, abs(количество ячеек) < 5.

Перечислите ячейки для цензурирования в поле Соседние строки или столбцы для цензурирования. Ячейки, которые будут цензурироваться, обозначаются своим положением по отношению к ячейке статистики цензурирования. Отрицательные числа должны быть слева от цензурируемой ячейки или над ней, положительные - справа или ниже. Включите 0 в список, чтобы цензурировать саму статистику цензурирования.

Например, если в таблице есть количества, средние значения и среднеквадратичные отклонения для столбцов, среднее и среднеквадратичное отклонение можно цензурировать на основании количества, вводя нужные значения
1 2
в списке.

Подтип OMS таблицы для обработки можно задать в поле Подтип OMS таблицы. Определить подтип можно, щелкнув правой кнопкой мыши по схематическому представлению для экземпляра или по идентификаторам утилиты/OMS. Регистр и пробельные символы не учитываются.

По умолчанию цензурирование будет применено к последней сводной таблице в программе просмотра. Можно указать Все, чтобы цензурировать все таблицы. Если задан подтип таблиц OMS, цензурирование будет ограничено таблицами этого типа.

Включите переключатель Скрыть строку или столбец критериев, чтобы скрыть всю строку или столбец, содержащий статистику цензурирования.

По умолчанию добавляется текст подписи, показывающий количество цензурированных ячеек и название статистики цензурирования. Чтобы этого не происходило, выключите переключатель Добавить подпись с информацией о цензурировании.

С помощью соответствующей радиокнопки можно управлять выводом этой подписи - она может выводиться всегда или же только в случае, когда какие-то ячейки действительно цензурируются.

Включите переключатель Пользовательский текст подписи и введите текст ниже, определяющий пользовательский текст подписи.

По умолчанию цензурированные ячейки становятся пустыми. Введите строку из одного или нескольких символов в поле Обозначение цензурированных ячеек, чтобы задать другой замещающий текст.

Предварительные требования

Требуется подключаемый модуль программирования Python. Его можно установить с установочного компакт-диска SPSS Statistics или скачать с сайта SPSS Developer Central (www.spss.com/devcentral).

Дополнительные возможности, доступные с помощью синтаксиса

При выполнении SPSSINC CENSOR TABLES /HELP создается полная синтаксическая диаграмма. В синтаксис можно включить команду, которая будет выполняться для создания таблицы в команде SPSSINC CENSOR TABLES. Это обеспечивает лучшее управление в случае неудачного исполнения команды генерирования таблицы.
